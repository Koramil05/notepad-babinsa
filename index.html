📋 CARA SETUP GOOGLE DRIVE:

  1. Buka https://console.cloud.google.com/
  2. Buat project baru
  3. Aktifkan Google Drive API
  4. Buat OAuth consent screen
  5. Buat credentials (OAuth client ID)
  6. Set Authorized origins ke URL GitHub Anda
  7. Copy Client ID dan paste di kode

📁 Dapatkan Folder ID:

  1. Buat folder di Google Drive
  2. Buka folder, lihat URL
  3. Contoh: https://drive.google.com/drive/folders/1abc123
  4. Folder ID = 1abc123
